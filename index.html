Portfolio

Bucket List

Description: A travel planning site where users can create highly customized itineraries, add photos and personal experiences and share their journeys with other users. This was a group project and I built the backend using Rails API with CRUD functionality and JSON web tokens for user authentication. The challenges in this project were securing the API endpoints and passing state changes from parent to child components and vice versa. Going forward, we intend to implement Redux to manage state changes and add testing.

Built with: React/Rails/PostgreSQL

Magic Mango

Description: A fresh take on DOTA2 strategy sites. Users can create and save hero builds depending on their playstyle and the current metagame. To obtain hero and item details, I fetched data from the DOTA2 API and displayed the images for users to click on. One of the challenges in this project was changing the contents of the form as users clicked on images to add or remove items. I solved this problem by adding hidden fields in the form.

Built with: JQuery/Node/Express/MongoDB

KryptoKnight

Description: A cryptocurrency trading tool that provides budding investors with technical analysis, a social trading feature and a virtual brokerage. This was a group project and I focused on the technical analysis feature. I fetched cryptocurrency data from the Cryptocompare API and manipulated the data before using MetricGraphics.js to perform charting. I got a lot of practice with object and array manipulation to calculate moving averages and plot multiple lines.

Built with: Ruby on Rails/PostgreSQL

Build-A-Burger

Description: My very first foray into coding. Users click on ingredients to make as many burgers as they can within the time limit. I initially had a lot of issues with the positioning of ingredients to overlap each other nicely, and with the layout of the starting and ending overlay. It was then that I discovered the magic of negative margins and z-index.

Built with: HTML/CSS/JS
